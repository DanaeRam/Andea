# Fantasy Grimoire UI

A high-quality Grimoire-themed UI kit designed for Fantasy RPG and Adventure games.

## 🛠 Getting Started

1. **TMP Import Required**: This asset requires **TextMeshPro (TMP)** to function correctly. Please ensure TMP is imported into your project.
2. **Demo Scene**: Open the **Demo_Scene** located in the `FantasyGrimoireUI/Demo` folder to preview the full UI layout and animations.

## 📦 Asset Contents

### 🎨 Icons (153 Count)
A comprehensive collection of fantasy item icons.
* **Armor**
* **Consumables**
* **Currency**
* **Materials**
* **Quest-Items**
* **Special-Items**
* **Tools**
* **Weapons**

### 📖 Book Animations (Sprite)
Sprite animation sequences for smooth book interactions.
* **Open/Close**: Book opening and closing motion 
* **Page Turn**: Page turning motion 

### 🧩 UI Elements
* **Buttons**: Basic buttons, Close button, Arrows, and Book icons (8 variations).
* **Decorations (Bookmarks)**: Tab buttons and decorative bookmarks.
* **Elemental Icons**: Elemental UI icons (12 variations).
* **Background**: Themed background image included.

### 🔤 Font
* **Google Noto Serif**: Includes Google Noto Serif (OFL - Open Font License) for excellent readability.
